# React Portfolio • GSAP + Framer Motion + Tailwind

A modern, animated portfolio starter featuring:
- Mouse‑reactive, moving color background (GSAP)
- Smooth section reveals (GSAP ScrollTrigger)
- Micro‑interactions and transitions (Framer Motion)
- Responsive Tailwind CSS layout
- Projects, Certifications, Skills, Contact, and Hero sections
- Mobile nav + sticky header

## Quickstart

```bash
npm i
npm run dev
# open the shown localhost URL
```

## Build & Preview
```bash
npm run build
npm run preview
```

## Customize
- Replace links, text, and items in:
  - `src/components/Hero.jsx` (name, socials, resume)
  - `src/data/projects.js` (projects)
  - `src/components/Certifications.jsx` (certs)
- Colors live in `tailwind.config.js` under `extend.colors`.
- Background animation: `src/components/AnimatedBackground.jsx`.

Enjoy!
